Web Development Exercise 06 - Input & Output

GREETINGS

This web page has used a common layout with content to specify the JavaScript test page. It will ask for the user's name by using the windows.prompt() command, and then assign the user input to a local variable. After that, it will return a message greeting the user by their name wuth the assistance of the windows.alert() command. Finally, it will return the same message from the user input to the console using a console.log() command.
